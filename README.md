# Mikrotik Hotspot Template

A futuristic Mikrotik Hotspot template built using Tailwind CSS.

### Requirement
- Node or Bun

### Setup dev

```bash
bun install
# or
npm install
```
### Start dev

```bash
bun dev 
# or
npm run dev
```

### Build

```bash
bun make
# or
npm run make
```
the result will be in the `results` folder. you can copy it to your mikrotik.
